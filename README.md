# 🔐 Password Manager

Простой консольный менеджер паролей на Python. Использует **гибридное шифрование**:
пароли шифруются симметричным ключом (Fernet / AES), а сам ключ — асимметричным
RSA-2048. Приватный RSA-ключ дополнительно защищён пароль-фразой.

Открытых паролей на диске не хранится: в `encrypted_passwords.json` попадают только
зашифрованные данные, а расшифровать их можно лишь имея `private_key.pem` **и**
пароль-фразу к нему.

## Возможности

- Генерация пары RSA-ключей, защищённой пароль-фразой
- Шифрование паролей любой длины (гибридная схема RSA + Fernet)
- Хранение записей в JSON с привязкой к сервису и логину
- Просмотр списка, расшифровка и удаление записей
- Скрытый ввод паролей в терминале (`getpass`)

## Как это устроено

```
пароль ──Fernet(случайный ключ)──▶ token
                    │
       случайный ключ ──RSA(публичный ключ, OAEP/SHA-256)──▶ encrypted_key

на диск сохраняются только token и encrypted_key
```

При расшифровке приватным RSA-ключом восстанавливается ключ Fernet, которым
расшифровывается сам пароль.

## Установка

Требуется Python 3.9+.

```bash
git clone https://github.com/<ваш-логин>/password-manager.git
cd password-manager

python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate

pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Появится меню:

```
--- Менеджер паролей ---
1. Сгенерировать пару ключей
2. Добавить и зашифровать пароль
3. Показать список записей
4. Расшифровать пароль
5. Удалить запись
6. Выход
```

### Порядок работы

1. **Пункт 1** — один раз сгенерируйте ключи и задайте пароль-фразу.
2. **Пункт 2** — добавьте пароль: укажите сервис, логин и сам пароль.
3. **Пункт 3** — посмотрите список сохранённых записей с их id.
4. **Пункт 4** — введите id записи и пароль-фразу, чтобы увидеть пароль.
5. **Пункт 5** — удалите ненужную запись.

## ⚠️ Безопасность

- **Никогда не коммитьте `private_key.pem` и `encrypted_passwords.json`** — они
  уже добавлены в `.gitignore`.
- Потеря приватного ключа или пароль-фразы означает, что расшифровать пароли будет
  **невозможно** — сделайте резервную копию ключа в надёжном месте.
- Это учебный проект. Для реального управления паролями рассмотрите проверенные
  решения (Bitwarden, KeePassXC, 1Password).
